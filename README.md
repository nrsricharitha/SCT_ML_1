# House Price Prediction using Linear Regression

## Overview

This project implements a Linear Regression model to predict house prices based on important housing features such as square footage, number of bedrooms, and number of bathrooms. The objective is to build a machine learning model that can estimate house prices accurately using historical housing data.

## Problem Statement

House prices depend on various factors, including property size and amenities. This project aims to develop a predictive model that learns the relationship between house features and their prices using Linear Regression.

## Dataset

The dataset contains information about houses, including:

* Square Footage
* Number of Bedrooms
* Number of Bathrooms
* House Price (Target Variable)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab / Jupyter Notebook

## Project Workflow

1. Import required libraries.
2. Load and explore the dataset.
3. Perform data preprocessing and cleaning.
4. Select relevant features.
5. Split the dataset into training and testing sets.
6. Train a Linear Regression model.
7. Evaluate model performance using regression metrics.
8. Visualize actual vs predicted house prices.

## Machine Learning Model

Linear Regression is a supervised learning algorithm used to predict continuous numerical values. In this project, it is used to estimate house prices based on housing characteristics.

## Evaluation Metrics

The model is evaluated using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

## Results

The trained model successfully predicts house prices based on the provided features. Performance is measured using standard regression evaluation metrics to assess prediction accuracy.

## Future Improvements

* Include additional housing features such as location and age of property.
* Experiment with advanced regression algorithms.
* Deploy the model as a web application.

## Conclusion

This project demonstrates the implementation of Linear Regression for house price prediction. It provides a simple and effective approach to understanding how machine learning can be applied to real-world real estate data.
